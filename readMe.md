
# Perintah Dasar File dan Direktori di Terminal
## Membuat dan Menulis
```
mkdir - Membuat folder baru
touch - Membuat file kosong baru
echo dan cat - Menulis teks ke dalam file
```

## Memindahkan dan Mengganti Nama
```
mv - Memindahkan file/folder atau mengganti namanya
```
## Menghapus
```
rm - Menghapus file
rmdir - Menghapus folder (harus kosong)
```
## Menyalin
```
cp - Menyalin file atau folder beserta isinya
```
